# TrustShield AI

## 🛡️ AI-Powered Fraud Prevention and Digital Trust Platform for East Africa

TrustShield AI is an AI-powered web application developed for the **EACA Summit Hackathon 2026** under the **Fraud Prevention & Trust** track.

The platform helps individuals, Small and Medium-sized Enterprises (SMEs), educational institutions, and organizations identify suspicious digital communications before they become victims of cyber fraud.

Instead of simply classifying content as malicious or safe, TrustShield AI explains the reasons behind its decision, helping users improve their cybersecurity awareness while building digital trust.

---

## Problem Statement

As East Africa rapidly adopts digital banking, mobile money, e-commerce, and online government services, cybercriminals increasingly exploit users through phishing emails, fraudulent SMS messages, malicious websites, Business Email Compromise (BEC), and other social engineering attacks.

Many existing security tools are either expensive, technically complex, or fail to educate users on why a message is considered dangerous.

TrustShield AI addresses this challenge by combining AI-powered fraud detection with explainable security recommendations.

## Installation

Clone the repository

```bash
git clone https://github.com/mading-KY/TrustSheld-AI.git
```

Navigate into the project

```bash
cd TrustSheld-AI
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm run dev
```
